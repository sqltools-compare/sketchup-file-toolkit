# sketchup-file-toolkit Documentation

Welcome to the documentation for sketchup-file-toolkit!

## Quick Start

```bash
git clone https://github.com/sketchup-tools/sketchup-file-toolkit
cd sketchup-file-toolkit
pip install -r requirements.txt
python main.py --help
```
