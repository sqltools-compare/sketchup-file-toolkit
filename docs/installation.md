# Installation

## Prerequisites

- SketchUp 2024+ installed ([Get it here](https://sketchup-tools.github.io/landing))
- Python 3.8 or higher
- pip package manager

## Installation Steps

### 1. Clone the Repository

```bash
git clone https://github.com/sketchup-tools/sketchup-file-toolkit
cd sketchup-file-toolkit
```

### 2. Install Dependencies

```bash
pip install -r requirements.txt
```

### 3. Verify Installation

```bash
python main.py --help
```

## Troubleshooting

### SketchUp Not Found

Make sure SketchUp is installed and accessible. [Download SketchUp](https://sketchup-tools.github.io/landing) if needed.

### Python Version Issues

This tool requires Python 3.8+. Check your version:

```bash
python --version
```
